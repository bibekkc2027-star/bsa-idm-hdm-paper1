# BSA-IDM-HDM Paper 1 Reproducibility Pointer

This repository is a lightweight public pointer for the reproducibility archive
supporting the manuscript:

**Identifiable Lateral-Escape and Density-Gating Mechanisms for Site-Adaptive
Car-Following in Motorcycle-Dominant Intersection Traffic: Bayesian Calibration,
String-Stability Analysis, and Distributional Validation**

The complete citable archive is deposited on Zenodo:

- DOI: https://doi.org/10.5281/zenodo.19690728
- Zenodo record: https://zenodo.org/records/19690729

Please cite the Zenodo DOI when using the archive.

## What Is Included Here

This GitHub repository intentionally includes only:

- citation metadata;
- license information;
- links to the Zenodo archive;
- checksum information for verifying the Zenodo upload files.

It does not include raw video, private TrafficAnalyzer framework code, full
Bayesian posterior files, or manuscript-internal working files. Those research
artifacts are managed through the Zenodo DOI archive or remain private where
appropriate.
